# SIMPLE CALCULATOR

Simple calculator is a website that can be used to calculate two numbers, such as addition, subtraction, multiplication and division.

## Features
- addition operation
- subtraction operation
- multiplication operation
- division operation

## Tech

* XAMPP 8.2.0 or later
* PHP 7.2 or later
